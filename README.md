## BLISH

```
Support_String::x86_64::GNU/Linux::Unix Makefiles
```

A mini shell for GNU/Linux like systems, written in C++.


### Building 

```bash
git clone https://github.com/aenlemmea/blish && cd blish/
bear -- make
```

### Running

```bash
./blish
```
